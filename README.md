# Java Practice and Learning Repository

This repository contains Java programs for beginners and students who are learning Java programming.

It includes chapter-wise notes, practice sets, examples, and problem-solving programs.
This repository is useful for learning Java basics, object-oriented programming (OOP),
and building strong programming fundamentals.

---

## About This Repository

This repository is created to help students and beginners understand Java step by step.

It contains:
- Basic Java programs
- Input and Output examples
- Operator and expression programs
- Control statement examples

Each folder contains:
- Java source code (.java files)
- README file with explanation

## How to Run Programs

1. Open the project in VS Code or any Java IDE.
2. Compile the file:

   javac FileName.java

3. Run the program:

   java FileName
